## Prerequisites

- [Node.js](https://nodejs.org/en/)

## Getting Started

```bash
git clone --depth=1 https://github.com/locol23/learn-react-typescript.git
cd learn-react-typescript
npm i
cd packages/<your_select_dir>
npm run dev
```

Go to [http://localhost:1234](http://localhost:1234)
